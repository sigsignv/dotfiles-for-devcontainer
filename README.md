# dotfiles for Dev Container

## Install

Add this to your VS Code user settings.  
(Note: this setting applies only to local user settings and does not work in Remote or devcontainer settings)

```json
{
  "dotfiles.repository": "sigsignv/dotfiles-for-devcontainer"
}
```
